# In this folder you can place your Intel microcodes.

# cpu[CPUID]_plat[Platform]_ver[Version]_[YYYY-MM-DD]_[Type]_[CRC32].bin
# Example:
# cpu50655_platB7_ver03000012_2019-04-12_PRD_50E6B5F2.bin

# After placing the microcodes you need, run the script GenUSRmC.bat.
